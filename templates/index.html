FIRST Results by Phone

At a FIRST Robotics competition it can be hard to get the latest match information without walking to the pits, or using a smartphone. Even though smartphones are everywhere, many of the primary participants in FIRST don't have smartphones.

Call or text the phone number 804-505-2659 to find out about the team.

Chances are you're reading this because someone on the Mathews Robotics team sent it to you. They're team 1137, and will be playing in the qualifying rounds on Friday, March 21 and Saturday, March 22. Normal text and phone call rates apply to using the service, but the number is supposedly a local Mathews phone number so that families can find out how the team is doing without calling long distance (though we encourage texting to get the most info).

SMS

Send a team number and we'll quickly respond with information about that team from the {{event.name}}.

Call

Call and follow the instructions. We'll read back to you almost everything we send in an SMS message.

Caveats

We don't have information about practice rounds (usually played on the Thursday of competition) because FIRST doesn't publish that information on the web.

Technical FAQs

I found a bug. Can you fix it?

Please file an issue on GitHub and include as much information as possible.

How do you interact with phones?

We use the twilio API.

Where does your data come from?

We get our initial data from The Blue Alliance, then pull data directly from FIRST during the matches (once every minute) to give you the most up to date information possible.

Can I get the source?

Absolutely. The project is hosted on GitHub. If you put something together that others might benefit from, please send a pull request and I'll do my best to get your code in the version I maintain.

I saw the source code and it's really ugly. Is this how you code?

No. I came up with this idea and decided to write it using some tools that were new to me (Heroku, Flask, twilio-python), in the week before the team I mentor (1137) went to regionals. Hopefully I'll get a chance to clean things up sometime soon.

Any other features planned?

I really want to have a basic web interface up and running before the Virginia Regional, but I can't promise that will happen.
